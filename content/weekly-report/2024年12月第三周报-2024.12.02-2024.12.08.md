```
Connection:
客户端 -- SNY Set >>     服务端
客户端 << SNY ACK Set -- 服务端
客户端 -- ACK Set >>     服务端

Send Data:
... 总是需要ACK回复的

Colse:
客户端/服务端 可能都会发送主动断开连接
服务端 -- FIN Set >>     客户端
服务端 << ACK FIN Set -- 客户端
服务端 -- ACK Set >>     客户端
服务端 << ACK Set --     客户端
```
